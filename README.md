# HireIQ - AI Resume Matcher

## Overview
HireIQ is an AI-powered Resume Matcher that analyzes resumes and compares them with job descriptions to help candidates improve their chances of getting shortlisted.

## Features
- Resume Upload
- Job Description Matching
- ATS Score Analysis
- Skill Gap Detection
- AI-Based Recommendations

## Tech Stack
### Frontend
- React.js
- Vite
- CSS

### Backend
- Node.js
- Express.js
